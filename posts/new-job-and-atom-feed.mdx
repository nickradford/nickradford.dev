## Atom Feed and Bring Back Blogging

Inspired by [Bring Back Blogging](https://bringback.blog/), I've opted to implement an atom feed here for my blog.

import { FontAwesomeIcon } from "@fortawesome/react-fontawesome";

[Here is the link to my feed <FontAwesomeIcon
  icon={faSquareRss}
  className="transition-colors text-zinc-500 dark:text-zinc-300 group-hover:text-sky-600 "

I built this using [jpmonette/feed](https://www.npmjs.com/package/feed) and referencing this [great blog post by Daan Debie](https://www.daan.fyi/writings/rss).

To power this, I have an API route at `api/feeds/atom.xml` which will

1. Create a feed using the `feed` library
2. Iterate through my posts using `feed.addItem` to add the relevant content (including the static markup for the post)
3. Set a `Cache-Control` header to 1 day
4. Return the xml
